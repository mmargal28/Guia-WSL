Recomiendo ver el tutorial o leer el apartado de comandos si no igualm instalaras algo que no quieres.
### Comandos para instalar wsl
```wsl --list --online```

```wsl set-default-version 2```

```wsl --install```

```wsl --install -d Ubuntu```

```wsl -l -v ```

```wsl --status```

```wsl --set-version Ubuntu 2```

```wsl --update ```
